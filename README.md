# Ninas ord

A little Swedish spelling game built for my daughter Nina.

**Play it: https://tgwizard.github.io/nina-ord/**

A word appears in gray outlines, and colorful letter bubbles float around
the play area. Tap the letters in order to fill in the word — each one
colors in its slot. Finish the word for confetti.

- 100 simple Swedish words
- Single HTML file, no build step, no dependencies
- Designed for ages ~4–6
